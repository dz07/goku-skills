## Installation

Clone this repo into your OpenClaw skills folder:

```bash
cd ~/.openclaw/workspace
git clone https://github.com/dz07/goku-skills.git skills
```

Or install individually via OpenClaw CLI:

```bash
openclaw skills install <skill-name>
```

## Verify

```bash
openclaw skills list
```

You should see all 48 skills available.
