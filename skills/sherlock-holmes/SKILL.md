# Sherlock Holmes Investigation Method

**MANDATORY:** Use for ALL complex investigations, debugging, analysis tasks.

## Core Protocol

1. **OBSERVE** - Collect only live, verifiable evidence
2. **DEDUCE** - Apply logical chains from known principles  
3. **ABDUCE** - Generate best-fit hypotheses
4. **TEST** - Verify with direct evidence
5. **DOCUMENT** - Record findings

## Evidence Standards
✅ **Accept**: Direct file reads, command outputs, API responses  
❌ **Reject**: Assumptions, "probably", cached memories

## Investigation Log Template
```markdown
## EVIDENCE
- [Verified fact 1]
- [Verified fact 2]

## HYPOTHESIS  
[Best explanation based on evidence]

## TEST
[Specific verification method]

## CONCLUSION
[Root cause with supporting evidence]
```

**No assumptions. Only facts.**
